# Usergrid Python SDK

# Overview
This is a starter project for the Usergrid Python SDK.  It is a work in progress.

# Installation

## PIP (http://pip.readthedocs.org/en/stable/installing/)

`pip install usergrid`

## Manual installation

- `git clone git@github.com:apache/usergrid-python.git`
- `cd usergrid`
- `pip install -e .`
